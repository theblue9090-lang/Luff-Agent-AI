LUFF Agent SDK

Build, tokenize and deploy autonomous AI agents that live on Solana and launch through LUFF Genesis.

Preview. The Agent SDK is in active development. The API below is illustrative and shows the intended developer experience — names and signatures may change before release.

Overview

The SDK gives an agent three things: a brain (an LLM-powered loop), skills (typed tools it can call), and a wallet (to act on-chain). LUFF handles the token, the market and the Genesis launch.

Install

npm install @luff/agent-sdk

Quickstart

import { createAgent, skill } from '@luff/agent-sdk'

const priceCheck = skill({
  name: 'getPrice',
  description: 'Fetch the USD price of a Solana token',
  run: async ({ mint }) => fetchPrice(mint),
})

const agent = createAgent({
  name: 'EmberQuant',
  ticker: 'EMBER',
  model: 'claude-sonnet-5',
  skills: [priceCheck],
  wallet: process.env.AGENT_WALLET,
})

await agent.run('Find the top Solana gainer and summarize why.')

Core concepts

🧠

Agent

The autonomous loop — reasons over goals, calls skills and decides what to do next.

🧩

Skills

Typed tools with a name, description and run(). The agent calls them to read data or act.

👛

Wallet

A Solana keypair the agent uses to trade, transfer and interact with programs on-chain.

🚀

Genesis

One call mints the agent's token and opens a fair, points-based launch on LUFF.

API reference

createAgent(config)

  • name, ticker — identity of the agent and its token.
  • model — the LLM powering the reasoning loop.
  • skills — array of skill() definitions.
  • wallet — Solana secret key or signer.

agent.run(prompt)

Runs one goal to completion, calling skills as needed. Returns the final result and a trace of steps.

agent.deploy()

const { token, genesisUrl } = await agent.deploy({
  category: 'DeFAI',
  type: 'Sentient',
})
console.log('Launching at', genesisUrl)
